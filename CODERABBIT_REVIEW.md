# Legal AI Service - CodeRabbit Review Plan

## 📋 Обзор проекта
Юридический AI сервис для генерации документов и анализа дел. Интеграция с Moonshot AI (Kimi API).

## 📁 Структура файлов

```
/opt/legal-ai-service/
├── app.py                  # Основной Flask app
├── auth.py                 # Аутентификация JWT
├── config.py              # Конфигурация
├── document_generator.py  # Генерация DOCX
├── kimi_api.py            # Интеграция Moonshot AI ⚠️ MOCK
```

## 🔗 Важные эндпоинты

| Метод | Эндпоинт | Описание | Статус |
|-------|----------|----------|--------|
| POST | `/api/auth/register` | Регистрация | ✅ |
| POST | `/api/auth/login` | Логин | ✅ |
| POST | `/api/cases` | Создание дела | ✅ |
| POST | `/api/cases/<id>/analyze` | AI анализ | ⚠️ MOCK |
| POST | `/api/cases/<id>/generate` | Генерация документа | ⚠️ MOCK |
| GET | `/api/balance` | Баланс кредитов | ✅ |

## ⚠️ Известные проблемы

1. **kimi_api.py** - работает в MOCK-режиме, нет реальных запросов к Moonshot AI
2. Нет обработки таймаутов при генерации документов
3. Нет rate limiting на API эндпоинтах

## 🎯 Цель
Подготовить сервис к production перед пополнением Moonshot AI кредитов.
