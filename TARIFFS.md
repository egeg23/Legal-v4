# 💼 ТАРИФЫ Legal AI Service

**Дата создания:** 17.03.2026  
**Статус:** Требуется реализация в коде

---

## 📋 Общие положения

- Система работает на **кредитах (запросах)**, не на подписке
- Пользователь покупает пакет → получает N запросов → тратит по мере использования
- При пополнении на определённую сумму автоматически активируется соответствующий тариф

## 🎯 Тарифы

### 1️⃣ Тариф «Разовый»

| Параметр | Значение |
|----------|----------|
| **Стоимость** | 5 000₽ |
| **Запросов** | 1 |
| **Документов на анализ** | до 3 |

**Особенности:**
- Целевая аудитория: люди, далёкие от юриспруденции
- **Бонус:** Отдельный документ «Разъяснение» — простым языком объясняется:
  - Что написал клиент
  - Что написал AI и почему именно так
  - Почему выбрана такая формулировка

---

### 2️⃣ Тариф «Адвокат»

| Параметр | Значение |
|----------|----------|
| **Стоимость** | 175 000₽ |
| **Запросов** | 50 |
| **Документов на анализ** | до 7 разом |

**Особенности:**
- Возможность внести «Ясность» по каждому документу
- Функция «Ясность»: юрист/адвокат указывает на ошибки AI, вносит корректировки
- AI учится на правках, сохраняет в базу знаний проекта

**Выгода vs Разовый:**
```
Разовый: 5 000₽ × 50 = 250 000₽ (если покупать по одному)
Адвокат: 175 000₽
Экономия: 75 000₽ (30%)
```

### 3️⃣ Тариф «Фирма»

| Параметр | Значение |
|----------|----------|
| **Стоимость** | 500 000₽ |
| **Запросов** | 175 |
| **Документов на анализ** | до 10 разом |

**Особенности:**

**Выгода vs предыдущих:**

### 4️⃣ Тариф «Палата»

| Параметр | Значение |
|----------|----------|
| **Стоимость** | 1 500 000₽ |
| **Запросов** | 500 |
| **Документов на анализ** | до 20 разом |
| **Внесений «Ясности»** | 10 раз в чате |

**Особенности:**
- Для каждого дела формируется **«Стратегия защиты подзащитного»**
- Аудит судебной практики по схожим делам из открытых источников
- Речь защиты
- Поиск коллизий
- Несколько сценариев с предиктивным исходом (% выигрыша дела)
- Все документы в формате Word

**Выгода vs предыдущих:**
```
Разовый: 5 000₽ × 500 = 2 500 000₽
Адвокат: 175 000₽ × 10 = 1 750 000₽
Фирма: 500 000₽ × 2.86 = 1 430 000₽
Палата: 1 500 000₽

vs Разовый: экономия 1 000 000₽ (40%)
vs Адвокат: экономия 250 000₽ (14%)
vs Фирма: разница всего 70 000₽, но + эксклюзивные фичи
Цена за запрос: 3 000₽
```

## 🔐 Регистрация и авторизация

### Процесс регистрации:
1. Пользователь вводит логин / пароль
2. Система отправляет письмо на почту со **случайным 4-значным кодом**
3. Код указан в теме и тексте письма
4. Пользователь вводит код на сайте для подтверждения

**Требование:** Письма отправляются с нашего сервера (не через внешние сервисы)

## 📊 История дел

Для **всех тарифов**:
- Каждый пользователь видит историю своих дел
- Сохраняются все документы, запросы, результаты

## 🔔 Система уведомлений

### Уведомления об остатке кредитов

| Остаток | Действие |
|---------|----------|
| < 25% | Email + всплывающее окно на сайте |
| < 15% | Email + всплывающее окно на сайте |
| < 10% | Email + всплывающее окно на сайте |
| < 5%  | Email + всплывающее окно на сайте |

### Скидка для новых клиентов

**Условие:** Оплатил первый раз, но ещё не оплатил второй

**Предложение:** Скидка 25% на тарифы 3 и 4 («Фирма» и «Палата»)

**Механика:**
- В уведомлении (email + поп-ап) содержится промокод/ссылка со скидкой
- Если пользователь оплачивает сумму минус 25% по этому уведомлению → тариф активируется
- Пример: «Фирма» за 375 000₽ вместо 500 000₽

## 💾 Технические требования (для разработки)

### Модели БД:

```python
class Tariff(db.Model):
    """Тарифы системы"""
    id = db.Column(db.Integer, primary_key=True)
    code = db.Column(db.String(20), unique=True)  # single, advocate, firm, chamber
    name = db.Column(db.String(50))
    price = db.Column(db.Integer)  # в копейках/рублях
    credits = db.Column(db.Integer)  # количество запросов
    max_documents = db.Column(db.Integer)  # документов за раз
    clarity_requests = db.Column(db.Integer)  # внесений ясности
    has_strategy = db.Column(db.Boolean)  # только для Палаты
    has_defense_speech = db.Column(db.Boolean)
    has_case_audit = db.Column(db.Boolean)

class User(db.Model):
    """Расширить существующую модель"""
    credits_balance = db.Column(db.Integer, default=0)
    current_tariff_id = db.Column(db.Integer, db.ForeignKey('tariffs.id'))
    tariff_activated_at = db.Column(db.DateTime)
    is_first_purchase = db.Column(db.Boolean, default=True)
    discount_used = db.Column(db.Boolean, default=False)

class ClarityRequest(db.Model):
    """Запросы на внесение ясности"""
    id = db.Column(db.Integer, primary_key=True)
    case_id = db.Column(db.Integer, db.ForeignKey('cases.id'))
    user_id = db.Column(db.Integer, db.ForeignKey('users.id'))
    lawyer_notes = db.Column(db.Text)  # правки юриста
    ai_learning_data = db.Column(db.Text)  # что AI запомнил
    created_at = db.Column(db.DateTime, default=datetime.utcnow)
```

### API Endpoints:

| Endpoint | Описание |
|----------|----------|
| `GET /api/tariffs` | Список тарифов с выгодой |
| `POST /api/balance/topup` | Пополнение баланса |
| `GET /api/balance` | Текущий баланс кредитов |
| `POST /api/cases` | Создание дела (-1 кредит) |
| `POST /api/clarity` | Внесение ясности |
| `GET /api/cases/history` | История дел пользователя |

- SMTP сервер настроен на нашем сервере
- Шаблоны писем для: подтверждения регистрации, низкий баланс, предложение скидки
