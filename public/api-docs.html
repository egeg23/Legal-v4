⚖️ Legal AI API

Генерация юридических документов: иски, апелляции, претензии, стратегии защиты.

POST /api/v1/legal/generate

Генерация юридического документа

Headers:

Content-Type: application/json
X-API-Key: wl_legal_live_your_key_here

Request Body:

{
  "document_type": "исковое_заявление",
  "description": "Взыскание задолженности по договору...",
  "claim_amount": 2500000,
  "documents": ["base64_encoded_file"]
}

Response:

{
  "success": true,
  "data": {
    "document_id": "doc_123456",
    "status": "generating",
    "remaining_requests": 98
  }
}

🛒 Seller AI API

Аналитика и оптимизация для продавцов на маркетплейсах (Wildberries, Ozon, Авито).

POST /api/v1/seller/analyze

Анализ карточки товара

Headers:

Content-Type: application/json X-API-Key: wl_seller_live_your_key_here

Request Body:

{
  "marketplace": "wildberries",
  "product_url": "https://www.wildberries.ru/catalog/12345/detail.aspx",
  "analysis_type": "full"
}

Response:

{
  "success": true,
  "data": {
    "analysis_id": "anl_789012",
    "competitors": [...],
    "recommendations": [...],
    "remaining_requests": 998
  }
}
