## 🔄 Процесс подачи заявки

### Этап 1: Подготовка (2-3 недели)
- [ ] Зарегистрировать ООО
- [ ] Подготовить все документы из списка
- [ ] Сделать презентацию проекта
- [ ] Записать демо-видео

### Этап 2: Регистрация на портале (1 день)
- [ ] Зайти на https://sk.ru
- [ ] Зарегистрировать компанию
- [ ] Заполнить профиль
- [ ] Загрузить учредительные документы

### Этап 3: Подача заявки (1-2 дня)
- [ ] Заполнить заявку на резидентство
- [ ] Загрузить техническое описание
- [ ] Загрузить бизнес-план
- [ ] Загрузить презентацию
- [ ] Указать команду

### Этап 4: Экспертиза (2-4 недели)
- [ ] Ожидание первичной проверки
- [ ] Возможно, запрос доп. документов
- [ ] Интервью с экспертами (онлайн)
- [ ] Презентация проекта (если потребуется)

### Этап 5: Получение статуса (1-2 недели)
- [ ] Положительное решение
- [ ] Подписание договора
- [ ] Внесение взноса
- [ ] Получение свидетельства резидента

## 💡 Полезные советы

### Увеличивает шансы:
1. **Работающий MVP** — покажи, а не расскажи
2. **Первые клиенты** — даже 1-2 реальных пользователя
3. **Команда** — чем больше разработчиков, тем лучше
4. **Патенты** — хотя бы заявка на патент
5. **Гранты** — подтверждение внешней экспертизы

### Уменьшает шансы:
1. ❌ Только идея, без реализации
2. ❌ Отсутствие команды (один учредитель)
3. ❌ Неинновационный продукт ("ещё один Uber")
4. ❌ Финансовые проблемы
5. ❌ Неполный пакет документов
