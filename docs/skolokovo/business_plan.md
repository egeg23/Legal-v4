## 2. Инновационность проекта

### 2.1. Проблема на рынке
- Высокая стоимость юридических услуг (от 30 000₽ за исковое заявление)
- Долгие сроки подготовки документов (3-7 дней)
- Недоступность качественной правовой помощи в регионах
- Сложность юридического языка для непрофессионалов

### 2.2. Решение
Автоматизация рутинных юридических задач с помощью ИИ:
- Снижение стоимости до 5 000₽ за документ
- Сокращение времени с дней до минут
- Доступность из любой точки России
- Два режима: юридический и "простым языком"

### 2.3. Технологические инновации
1. **NLP-модель** для понимания юридических текстов
2. **Генеративный AI** для создания документов
3. **RAG-архитектура** (Retrieval-Augmented Generation) с базой судебной практики
4. **Мультимодальность** — обработка PDF, DOCX, изображений
5. **Адаптивные шаблоны** под разные суды и регионы

### 2.4. Отличие от конкурентов

| Параметр | Конкуренты | Legal AI |
|----------|------------|----------|
| Стоимость | 30 000-50 000₽ | 5 000₽ |
| Скорость | 3-7 дней | 2-5 минут |
| Доступность | Офис, запись | 24/7 онлайн |
| Язык | Юридический жаргон | + "Простым языком" |
| Масштабирование | Линейное | Экспоненциальное |

## 3. Продукт

### 3.1. Функционал

**Для пользователей (тариф "Разовый"):**
- Загрузка до 5 документов
- Выбор типа документа (чекбоксы) или свободный запрос
- Генерация в формате Word
- Объяснение "простым языком"

**Для юристов (тариф "Адвокатский"):**
- 50 запросов/месяц
- До 10 документов на запрос
- История обращений
- Приоритетная обработка

**Для адвокатских палат (тариф "Адвокатская палата"):**
- 300 запросов/месяц
- До 20 документов на запрос
- **История дел по клиентам**
- Внутренний поиск по делам
- Функция оценки документов

### 3.2. Технологический стек
- **Backend:** Python, Flask, SQLAlchemy
- **AI:** Moonshot AI (Kimi) API
- **База данных:** SQLite (MVP), PostgreSQL (production)
- **Frontend:** HTML5, CSS3, JavaScript
- **Документы:** python-docx
- **Инфраструктура:** Ubuntu, Gunicorn, Nginx

### 3.3. Этапы разработки

## 4. Маркетинг и продажи

### 4.1. Целевой рынок

**TAM (Total Addressable Market):**
- Юридические услуги в РФ: ~300 млрд₽/год
- Онлайн-сегмент: растёт на 25% в год

**SAM (Serviceable Addressable Market):**
- Мелкие юридические услуги (до 50 000₽): ~50 млрд₽
- Целевая доля: 0.1% = 50 млн₽

**SOM (Serviceable Obtainable Market):**
- Год 1: 1 000 клиентов × 5 000₽ = 5 млн₽
- Год 2: 5 000 клиентов = 25 млн₽
- Год 3: 20 000 клиентов = 100 млн₽

### 4.2. Каналы привлечения
1. **SEO:** запросы "исковое заявление онлайн", "составить иск"
2. **Контекстная реклама:** Яндекс.Директ
3. **Партнёрства:** с юридическими сайтами (Право.ру, Гарант)
4. **B2B:** прямые продажи адвокатским палатам
5. **Telegram:** каналы для юристов

### 4.3. Юнит-экономика

**Стоимость привлечения (CAC):**
- B2C: ~500₽ (контекст + SEO)
- B2B: ~5 000₽ (прямые продажи)

**Lifetime Value (LTV):**
- Тариф "Разовый": 5 000₽
- Тариф "Адвокатский": 150 000₽
- Тариф "Палата": 500 000₽

**LTV/CAC > 10** — отличная экономика
